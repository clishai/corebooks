corebooks open source self-hosted free forever

Your books.
Your machine.

Open-source, privacy-first accounting software. No subscription, no cloud account, no data leaving your device — ever.

Get started

Requires Node.js 20+ and Git. No other software needed — SQLite is bundled. 

# 1. clone the repo
git clone https://github.com/clishai/corebooks.git
cd corebooks

# 2. install dependencies
npm install

# 3. start the app (API + UI together)
npm run dev

Once running, open http://localhost:5173 in your browser. corebooks will walk you through setup on first launch.

Double-entry bookkeeping

Chart of accounts, journal entries, trial balance, balance sheet, and income statement — built on a pure accounting engine.

Privacy first

Everything lives on your machine. No analytics, no telemetry, no cloud account required — ever.

Encrypted backups

Export your data encrypted with AES-256-GCM. Your passphrase, your backup. No recovery if you lose the passphrase.

Multi-user ready

Connect a PostgreSQL server for teams that need shared access across multiple devices simultaneously.

Draft entries

Work in progress is saved automatically. Entries are only posted when debits equal credits exactly — no exceptions.
